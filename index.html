System Layers (The “Smart-Flat” Architecture)

The flat, geo-tagged stack that keeps LinkNode responsive.

Presentation Layer (React.js + Tailwind)

Mapbox GL JS renders the 3D globe and draws trade arcs between saved coordinate pairs. Dashboards surface health pie charts, risk gauge meters, and comparison cards for alternative suppliers.

  • Mapbox GL JS globe with live anchor-supplier nodes
  • Risk Dashboard pies and gauges for situational awareness
  • Action Center cards contrasting at-risk vs. alternative suppliers

Logic Layer (Node.js + Express)

Tag-processor handles status updates for entire anchor_tag groups while the API gateway serves flat GeoJSON data and calculates buffer-day delay math on demand.

  • Anchor_tag processor for instant group status flips
  • GeoJSON pipeline powering globe and dashboards
  • Notification engine that reacts when Sentinel flags locations

Intelligence Layer (Python)

The Sentinel scrapes global news, extracts city-level metadata, and geo-codes events so disruption signals map directly back to the flat supplier dataset.

  • News + weather scrapers crawling global feeds
  • NLP keyword and city extraction
  • Geo-coding utilities aligning events with supplier coordinates

Data Layer (MongoDB Atlas)

Flat schema stores parentCoords and anchorTag on every supplier document, while risk logs archive a rolling 30-day history of disruption events per city.

  • Flat supplier documents with parentCoords and anchorTag
  • Risk log collection indexed by city and timestamp
  • Backup snapshots for compliance-ready auditing

Real-Time Scenario: Surat Textile Strike

How a local disruption becomes a global pivot in minutes.

  1. Detection (00:00) — Python Sentinel flags “Fabric Strike in Surat” using Currents News API.
  2. Mapping (00:02) — Node.js runs updateMany({ city: "Surat" }, { status: "Red" }) across flat supplier docs.
  3. Propagation (00:05) — Anchor tag (e.g., “Aura Apparel”) groups are identified and bufferDays summed (e.g., 12 days).
  4. Mitigation (00:10) — Plan B finder recommends a Green supplier in Coimbatore with matching material.
  5. Result — 12 days of delay avoided, 10 minutes to pivot sourcing.
