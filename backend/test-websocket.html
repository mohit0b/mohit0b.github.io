‚ö° Quick REST API Test (cURL Commands)

1. Admin Login:

curl -X POST http://localhost:3000/api/v1/auth/login ^
-H "Content-Type: application/json" ^
-d "{\"email\":\"admin@linknode.com\",\"password\":\"adminPassword\"}"

2. Create Shipment:

curl -X POST http://localhost:3000/api/v1/shipments ^
-H "Authorization: Bearer YOUR_JWT_TOKEN" ^
-H "Content-Type: application/json" ^
-d "{\"origin_address\":\"123 Delhi\",\"destination_address\":\"456 Mumbai\",\"driver_id\":2}"

3. Driver Sends GPS (Triggers Real-Time Update):

curl -X POST http://localhost:3000/api/v1/tracking/update ^
-H "Authorization: Bearer DRIVER_JWT_TOKEN" ^
-H "Content-Type: application/json" ^
-d "{\"shipment_id\":1,\"latitude\":28.6139,\"longitude\":77.2090}"
